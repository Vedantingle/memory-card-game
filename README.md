# 🃏 Memory Card Game  

The Memory Card Game is a fun and interactive web-based project that challenges players to test and improve their memory skills. Flip the cards, find matching pairs, and win the game in as few moves as possible. This game is simple, engaging, and perfect for users of all ages!  

## 🎮 Features  
- **Interactive Gameplay** – Flip cards and match pairs.  
- **Timer & Moves Counter** – Track your progress with a timer and move counter.  
- **Responsive Design** – Fully functional on desktop, tablet, and mobile.  
- **Engaging UI** – Clean and modern design for a smooth user experience.  
- **Restart Option** – Reset the game anytime to start fresh.  

## 🛠️ Tech Stack  
- **HTML5** – Structure and layout.  
- **CSS3** – Styling and animations.  
- **JavaScript** – Game logic and interactivity.  

## 🚀 Installation  
1. Clone this repository:  
   ```bash
   git clone https://github.com/yourusername/memory-card-game.git  
   ```  
2. Navigate to the project directory:  
   ```bash
   cd memory-card-game  
   ```  
3. Open `index.html` in your browser.  

## 🗂️ File Structure  
- **index.html** – Main game interface.  
- **style.css** – Game styling and animations.  
- **script.js** – Core game logic.  

## 🧩 How to Play  
1. Click on any card to flip it over.  
2. Find and flip another card to find a matching pair.  
3. Match all pairs to win the game.  
4. Reset and try to beat your best time!
